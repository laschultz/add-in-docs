
# Slice.index property (JavaScript API for Office)
Gets the index of the file slice.

|||
|:-----|:-----|
|**Hosts:**|PowerPoint, Word|
|**Available in [Requirement set](http://msdn.microsoft.com/library/6b6702f2-b0a5-46ab-a356-8dda897ca8ae%28Office.15%29.aspx)**|File|
|**[Last changed](#bk_history) in**|1.1|
[See all support details](#bk_support)

```
var sliceIndex = slice.index;
```


## Return value

The zero-based index of the file slice.
